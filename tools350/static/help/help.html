Help Quickstart

Some basic pointers that might help with common errors
  • You may only have one jump/branch target per line. Because of how the assembler works, though, you can place a target-only line right after your first target and they'll both map to the same line.
  • There is no need for the normal MIPS structure with data and text fields. Simple write the instructions you want into a file directly. See the examples in the documentation .
  • Labels cannot be purely numeric, and they cannot they use any of these:
    ( ) : ; $ ,
  • The assembler does no error checking on your supplemental files. Make sure they adhere to the spec laid out in the documentation
  • If the assembler gives you a "KeyError", it likely means you attempted to use an instruction that it didn't recognize. Make sure that your code is correct and try again.
